Twee to HTML (build)
    Example command: ./tweego_folder/tweego -l -o export/test.html src

Twee to HTML (debug build)
    Example command: ./tweego_folder/tweego -t -l -o export/Debug.index.html src

HTML to Twee (decompile)
    Example command: ./tweego_folder/tweego -d test.html -o src/test.tw
